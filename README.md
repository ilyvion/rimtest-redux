RimTest Redux is a RimWorld mod that adds a lightweight automated testing framework for other mods. It lets modders write and register test suites that run in-game, with a settings UI to trigger runs and inspect pass/fail/warning status per assembly, test suite, and individual test.

Originally created by LaTrissTitude as RimTest; this is a continuation/fork.
